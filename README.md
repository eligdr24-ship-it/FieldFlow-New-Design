# FieldFlow / TechLink Marketplace MVP

A mobile-responsive MVP that positions the product as a Fiverr-style marketplace for field-service professionals, with FieldFlow operations layered on top.

## Run locally
```bash
npm install
npm start
```
Open http://localhost:3000

## Deploy on Render
- Build command: `npm install`
- Start command: `npm start`
- Environment: Node 18+

## Included screens
- Discover marketplace homepage
- Technician profile page with Availability / Performance / Coverage trust card
- Publisher dashboard for finding techs and broadcasting partial lead info
- Technician Field Dashboard / mobile app view
- Active Jobs Board with expandable sections
- Tech Network posts: techs looking for work, companies hiring, publishers seeking crews
- Payout reconciliation table

## Mobile responsive
- Sticky mobile header
- Horizontal category navigation
- Slide-out mobile menu
- Single-column mobile cards
- Responsive technician field dashboard
